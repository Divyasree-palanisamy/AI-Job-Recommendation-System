# Smart Career Guidance & Employment Portal - Project Tracker

## Project Description

**Smart Career Guidance & Employment Portal** is a comprehensive web-based platform that provides personalized career guidance, job market insights, and employment support for students and job seekers. The platform leverages AI-powered recommendations, real-time market analytics, and interactive tools to help users make informed career decisions and achieve their professional goals.

### Admin Side Features & Tech Stack
- **User Management**: Admin dashboard for managing student profiles, analytics, and system monitoring
- **Content Management**: AI chatbot configuration, job market data updates, analytics reporting
- **System Administration**: Database management, user authentication oversight, platform monitoring
- **Analytics Dashboard**: Career trend analysis, student performance metrics, platform usage statistics

**Tech Stack**: Python Flask, SQLite Database, HTML/CSS/JavaScript, Chart.js for analytics, Admin authentication system

### Student Side Features & Tech Stack
- **Personalized Dashboard**: Career progress tracking, skill assessment, job recommendations
- **AI Career Chatbot**: 24/7 AI-powered career guidance and interview preparation
- **Job Market Intelligence**: Real-time salary data, industry trends, skill demand analytics
- **Resume Builder**: Dynamic resume generation with ATS optimization
- **Profile Management**: Skills tracking, career goals setting, progress monitoring
- **Interactive Learning**: Course recommendations, skill gap analysis, certification tracking

**Tech Stack**: HTML/CSS/JavaScript (Frontend), Python Flask (Backend), SQLite Database, AI/ML integration for recommendations, Responsive design with mobile PWA support
