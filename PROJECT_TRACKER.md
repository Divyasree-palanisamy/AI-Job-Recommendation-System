# Smart Career Guidance & Employment Portal - Project Tracker

## Project Description

**Smart Career Guidance & Employment Portal** is a web-based platform that provides personalized job recommendations and career support for students. The platform uses AI-powered job matching, resume analysis, and learning resources to help students find suitable career opportunities.

### Admin Side Features & Tech Stack
- **User Management**: Admin dashboard for managing student profiles and system content
- **Content Management**: Job postings, courses, videos, and student data management
- **System Administration**: Database management and user authentication oversight

**Tech Stack**: Python Flask, SQLite Database, HTML/CSS/JavaScript, Admin authentication system

### Student Side Features & Tech Stack
- **Personalized Dashboard**: Student profile management and career tracking
- **AI Job Recommendations**: ML-powered job matching based on student profiles
- **Resume Analysis**: ATS-compatible resume upload and basic analysis
- **Learning Resources**: Course and video recommendations for skill development
- **Profile Management**: Skills, experience, and career goals tracking

**Tech Stack**: HTML/CSS/JavaScript (Frontend), Python Flask (Backend), SQLite Database, Scikit-learn for AI recommendations
