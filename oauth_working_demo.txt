3. **Redirects to OAuth route** (`/auth/google`)
4. **Demo mode generates fake user data**
5. **process_oauth_user() function:**
   - Checks if email exists in database
   - If exists: logs in user
   - If new: creates account with unique username
   - Sets session variables
   - Shows welcome message
   - Redirects to dashboard

## üöÄ PRODUCTION SETUP:

When ready for production, set `DEMO_OAUTH_MODE = False` and:

1. **Get real OAuth credentials:**
   - Facebook: https://developers.facebook.com/
   - Google: https://console.cloud.google.com/

2. **Update .env file:**
   ```
   FACEBOOK_APP_ID=your_real_app_id
   FACEBOOK_APP_SECRET=your_real_secret
   GOOGLE_CLIENT_ID=your_real_client_id
   GOOGLE_CLIENT_SECRET=your_real_secret
   ```

3. **Configure redirect URIs:**
   - Facebook: `https://yourdomain.com/auth/facebook/callback`
   - Google: `https://yourdomain.com/auth/google/callback`
